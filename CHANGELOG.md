# Changelog

All notable changes to **CalcBoy** are documented here.

## Version 1.7

### Added

-   Formula Assistant
-   RPN mode
-   History PNG export
-   Seventh console theme (unlockable Virtual Boy)

### Changed

-   Finance **INFO** now shows end value, total paid in and interest
    earned.
-   Improved iPhone PWA Safe Area support so top controls are not hidden
    by the status bar.
-   Various UI improvements and bug fixes.
